Freytag Forge is a Python 3.12 deterministic interactive-fiction engine with CLI and FastAPI web entrypoints. Core package is `storygame/` with engine, plot, llm, persistence, and web modules. Existing local web UI is implemented in `storygame/web.py`. Product and architecture notes live in `docs/PRD.md` and implementation plans in `.plans/`. Tech stack: FastAPI, Uvicorn, Pydantic, PyYAML, pytest, Ruff, uv.
